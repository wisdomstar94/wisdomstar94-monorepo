# @wisdomstar94/\* 라이브러리들

npm 저장소에 `@wisdomstar94` 네임스페이스로 시작하는 여러 라이브러리들을 배포하였습니다.

해당 라이브러리들은 [wisdomstar94-monorepo](https://github.com/wisdomstar94/wisdomstar94-monorepo) 레포지토리에서 개발 및 유지보수 하고 있습니다. 현재 react 프레임워크인 next.js 를 사용하여 실무에서 프론트엔드 개발을 하고 있어서 react 환경에서 사용 가능한 패키지를 많이 개발 및 배포한 상태입니다.

개인적으로 `rust` 에도 관심이 있어서 rust 로 만든 라이브러리 및 프레임워크도 npm 저장소에 배포한 것도 있습니다.

다양한 종류의 라이브러리가 있으니 필요한 게 있으시면 골라 사용하시면 되겠습니다. 만약 특정 라이브러리에 버그나 요청사항 등이 있다면 https://github.com/wisdomstar94/wisdomstar94-monorepo/issues 으로 이슈를 남겨주시면 확인해보도록 하겠습니다.
